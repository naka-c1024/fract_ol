fract-ol
====

##

## clone & install

```
git clone <URL>
cd fract_ol
make
```

After running make, you can see fractol of an executable file.

## Usage

### How to use program

```
./fractol
```

### Mandatory part

- Mandelbrot set

```
./fractol Mandelbrot
```

- Julia set

```
./fractol Julia
```

### Bonus part
- Burning ship
```
./Burning_ship
```
